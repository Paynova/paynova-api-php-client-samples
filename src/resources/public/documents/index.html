Paynova API PHP Client

Sample site

1 Edit the properties file

For this sample site to work you need to contact Paynova support to get URL, username and password to the test enviroment. Then edit the{code-snippet}paynova-samples.properties{/code-snippet}

2 callback-store-file.txt

Make sure that {code-snippet}src/resources/callback-store-file.txt{/code-snippet} is writable by the webserver to be able to store Event Hook Notifications

3 Starting point

Start by Creating an order and then Initialize Payment. Then explore the other services.

4 Sample Code

All sample pages shows code. If you want to take a look at the real sample code take a look at classes in: {code-snippet}src/classes/document{/code-snippet}
