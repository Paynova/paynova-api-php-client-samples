Annul Authorization

After a successful customer payment process (in the payment window), the transaction can get the status authorized. Here you can annul that authorization

Before Annul Authorization can take place the following the following must have occurred:

  1. An order must have been created and payment initialized
  2. Customer must have paid using credit card or other payment method that results in a transaction with the status authorized

Note if you get error message from the API server, the transaction might already have been finalized. Go to create order and initialize payment.
