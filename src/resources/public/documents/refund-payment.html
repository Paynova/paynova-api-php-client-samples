Refund Payment

The Refund Payment service is used to refund a Payment within Paynova's system.

Before Refund Payment can take place the following must have occurred:

  1. An order must have been created and payment initialized
  2. Customer must have paid
  3. The payment must be in a completed state. If credit card was used as payment method, Finalize Authorization have to be called

Note if you get error message from the API server, the transaction might already have been refunded. Go to create order and initialize payment.
